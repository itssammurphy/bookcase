# Bookcase: A Private, Synced Home for Every Book She Loves

Bookcase is a small hobby project of mine developed in [Next.js](https://nextjs.org) to help my girlfriend and I track our personal reading.

## Core features

- Add, edit, remove books
- Add books to series with chronological order
- Track series completion %
- Track authors (ratings, number of books, etc)
- Tag your books with genres, etc
- Create custom lists
- Track read, unread, and reread books
